# AW CLI

## Install

```bash
$ git clone git@github.com:unkleho/aw-cli
$ npm install
$ npm run dev
$ npm start
```

## Setup

```zsh
# In your ~/.zshrc file
alias aw="node /Users/path/to/aw-cli/dist/cli.js $1 $2"
```

## Usage

```zsh
$ aw
# Will show Git and Test options

$ aw git
# Show Git options

$ aw test
# Show test options
```
